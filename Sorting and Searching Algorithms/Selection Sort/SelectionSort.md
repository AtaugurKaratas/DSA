# Selection Sort

1. [7, 3, 5, 8, 2, 9, 4, 15, 6]

2. [**`2`**, 3, 5, 8, **`7`**, 9, 4, 15, 6]

3. [2, 3, **`4`**, 8, 7, 9, **`5`**, 15, 6]

4. [2, 3, 4, **`5`**, 7, 9, **`8`**, 15, 6]

5. [2, 3, 4, 5, **`6`**, 9, 8, 15, **`7`**]

6. [2, 3, 4, 5, 6, **`7`**, 8, 15, **`9`**]

7. [2, 3, 4, 5, 6, 7, 8, **`9`**, **`15`**]

- Best Case: **`O(n²)`**
- Average Case: **`O(n²)`**
- Worst Case: **`O(n²)`**
